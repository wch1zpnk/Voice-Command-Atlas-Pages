Privacy

Your command data stays on your Mac.

Voice Command Atlas is designed for local command review and maintenance. It does not upload your custom Voice Control command contents.

What The App Reads

The Mac App Store version reads only the .voicecontrolcommands export you explicitly choose in a standard macOS Open dialog.

When The App Changes Data

The app changes custom command data only after explicit actions such as add, edit, delete, enable, disable, restore, refresh reference, or bulk profile changes. In the Mac App Store version, those changes affect only the selected export until you re-import it through Voice Control's Options menu.

Backups

Before changing command data, Voice Command Atlas saves a backup snapshot under Application Support:

~/Library/Application Support/Voice Command Atlas/Backups

Network Use

Command data, snapshots, backups, and command test results stay local. The app does not send custom command contents to JBDApps or any remote service.
