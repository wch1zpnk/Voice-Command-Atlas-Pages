Support

Help with Voice Command Atlas.

For support, setup questions, or bug reports, email support@jbdapps.com.

Requirements

  • macOS 14 or newer
  • Voice Control custom commands created in macOS Accessibility settings

Common Checks

  • Open Compatibility Check in the app to confirm the command file and backup location are available.
  • Use Health to review duplicates, disabled commands, missing files, and stale bookmarks.
  • Use Backups before restoring or making broad command changes.

Voice Control Reloads

After command changes, the app may restart Voice Control helper processes so macOS reloads your saved commands. This keeps the visible command state and the active speech grammar aligned.
