Support

Help with Voice Command Atlas.

For support, setup questions, or bug reports, email support@jbdapps.com.

Requirements

  • macOS 14 or newer
  • Voice Control custom commands created in macOS Accessibility settings

Common Checks

  • Open Compatibility Check in the app to confirm the selected export and backup location are available.
  • Use Health to review duplicates, disabled commands, missing files, and stale bookmarks.
  • Use Backups before restoring or making broad command changes.

Using The Mac App Store Version

In System Settings > Accessibility > Voice Control, open the Options menu and choose Export Custom Commands. Choose that .voicecontrolcommands file in Voice Command Atlas. After editing it, return to the same Options menu and choose Import Custom Commands to apply the changes.

Reopening The Main Window

If the main window is closed while the app is still running, choose Window > Show Voice Command Atlas Window or press Command-1.
