macOS Voice Control utility

Find, check, and maintain your custom Voice Control commands.

Voice Command Atlas helps you search custom commands, spot duplicate phrases, test saved actions, refresh stale file references, and keep restore points before making changes.

Command Review

Search by phrase, app scope, action type, menu item, shortcut, target file, or resolved file path.

Health Checks

Review duplicates, disabled commands, stale bookmarks, missing files, broad menu scopes, and risky shortcuts.

Safer Changes

Create backups before edits, restores, deletes, status changes, bookmark refreshes, and bulk profile changes.

Built For Local Voice Control Data

The Mac App Store version works with the custom-command export you select. It reads command data locally and changes that file only after explicit in-app actions; re-import the edited export in Voice Control to apply it.
